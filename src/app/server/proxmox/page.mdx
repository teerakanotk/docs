# Proxmox VE

Proxmox VE (Proxmox Virtual Environment) คือแพลตฟอร์มการจัดการเซิร์ฟเวอร์แบบเสมือนจริง (Server Virtualization Management Solution) แบบโอเพนซอร์ส (Open Source) ที่ทรงพลังและใช้งานง่าย

อธิบายง่ายๆ คือ Proxmox VE เปลี่ยนเครื่องคอมพิวเตอร์เซิร์ฟเวอร์จริง ๆ ของคุณให้กลายเป็น "โฮสต์" ที่สามารถรันระบบปฏิบัติการและแอปพลิเคชันเสมือนจริงได้หลายตัวพร้อมกัน หรือที่เรียกว่า Virtual Machine (VM) และ Container (LXC) โดยใช้ทรัพยากรของเซิร์ฟเวอร์จริงร่วมกันอย่างมีประสิทธิภาพ

## วิธีการโหลดอิมเมจไฟล์

<Steps>

### เปิดเว็บเบราว์เซอร์และไปที่

```
https://www.proxmox.com/en/downloads
```

### เลื่อนลงมายังหัวข้อ Proxmox VE ISO Installer

เลือกเวอร์ชันที่ต้องการ จากนั้นคลิก **Download**

![proxmox-boot-1](/image/proxmox/proxmox-boot-1.png)

</Steps>

## ขั้นตอนการติดตั้ง Proxmox VE

<Steps>

### Welcome to Proxmox Virtual Environment

- เลือก **Install Proxmox VE**
- กดปุ่ม **Enter**

![proxmox-setup-1](/image/proxmox/proxmox-setup-1.png)

### END USER LICENSE AGREEMENT (EULA)

อ่านข้อตกลงการใช้งาน (End-User License Agreement) เพื่อทำความเข้าใจเงื่อนไขการใช้งาน

- คลิก **I agree** เพื่อยอมรับข้อตกลงและไปยังขั้นตอนถัดไป

![proxmox-setup-2](/image/proxmox/proxmox-setup-2.png)

### Proxmox Virtual Environment (PVE)

เลือกดิสก์ที่ต้องการติดตั้ง แล้วคลิก **Next**

![proxmox-setup-3](/image/proxmox/proxmox-setup-3.png)

### Location and Time Zone selection

- Country: **ระบุ Country**

- Time Zone: **เลือก Timezone**

- Keyboard Layout: **เลือก Layout**

- คลิก **Next**

![proxmox-setup-4](/image/proxmox/proxmox-setup-4.png)

### Administration Password and Email Address

- Password: **กรอกรหัสผ่านที่คุณต้องการใช้สำหรับบัญชีผู้ดูแลระบบ**
- Confirm Password: **กรอกรหัสผ่านซ้ำอีกครั้งเพื่อยืนยัน**
- Email: **กรอกอีเมลสำหรับแจ้งเตือนหรือการกู้คืน**

> หากไม่ต้องการใส่อีเมลส่วนตัว ให้ระบุเป็น `mail@example.local`

![proxmox-setup-5](/image/proxmox/proxmox-setup-5.png)

### Management Network Configuration

- Hostname: **ใส่ชื่อเครื่อง**
- IP Address: **ใส่ IP Address**
- Netmask: **ใส่ Subnet Mask**
- Gateway: **ใส่ Gateway**
- DNS Server: **ใส่ DNS Server**

![proxmox-setup-6](/image/proxmox/proxmox-setup-6.png)

### Summary

คลิก **Install** เพื่อเริ่มการติดตั้ง Proxmox VE

![proxmox-setup-7](/image/proxmox/proxmox-setup-7.png)

### เข้าสู่ระบบครั้งแรก

- เปิดเว็บบราวเซอร์ของคุณ แล้วพิมพ์ **IP Address** ที่แสดงบนหน้าคอนโซล ตามด้วยพอร์ต **8006**

```url copy
https://<ip_address>:8006
```

> แทนที่ `<ip_address>` ด้วยที่อยู่ IP ของเซิร์ฟเวอร์คุณ

```
Username: root
Password: <your_password>
```

</Steps>
