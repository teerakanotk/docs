# Zabbix

## ขั้นตอนการติดตั้ง Zabbix Server บน Linux Ubuntu 22.04 LTS

<Steps>
### โหลดสคริปต์การติดตั้ง Zabbix อัตโนมัติจาก Github repository

stack ที่ใช้งาน
- zabbix Server
- zabbix agent2
- postgresql-17
- nginx

```bash copy
wget https://github.com/teerakanotk/automated-script/raw/main/zabbix-pgsql-nginx-install.sh
```

### รันสคริปต์ติดตั้ง

```bash copy
sudo sh zabbix-pgsql-nginx-install.sh
```

### ล็อกอินเข้าสู่ Zabbix ครั้งแรก

- เปิดเว็บเบราว์เซอร์ แล้วพิมพ์ URL ที่แสดงบนหน้า console:

```bash copy
http://<server_ip_address>/
```

- คลิก `Next step`

![zabbix-install-1](/server/zabbix/zabbix-install-1.png)

- เช็คสถานะของ module เป็น `OK` ทั้งหมดหรือยัง หากครบจากนั้นคลิก `Next step`

![zabbix-install-2](/server/zabbix/zabbix-install-2.png)

- ตั้งค่า Database
    - Password: นำรหัสผ่านที่แสดงบนหน้าจอ Console มาใส่ในช่องนี้
    - ตัวอย่าง: 
        - Password: `<strong_password>`

- คลิก `Next step`

![zabbix-install-3](/server/zabbix/zabbix-install-3.png)

    - Default theme: `<ไม่บังคับ>`
