# Technitium

Technitium คือซอฟต์แวร์ DNS Server แบบโอเพนซอร์ส (Open Source) ที่มีความสามารถหลากหลาย ถูกออกแบบมาให้ ใช้งานง่ายผ่าน Web-based Dashboard โดยรองรับทั้งการทำงานแบบ Authoritative DNS (จัดการโซน DNS ของตัวเอง) และ Recursive Resolver (สอบถาม DNS จากเซิร์ฟเวอร์ภายนอก)

**Features:**
- รองรับโปรโตคอล DNS แบบเข้ารหัส (DNS-over-TLS, DNS-over-HTTPS, DNS-over-QUIC) เพื่อปกป้องข้อมูล DNS ของคุณจากการสอดแนม
- สามารถสร้าง Blocklist เพื่อบล็อกโฆษณาและเว็บไซต์อันตรายได้
- รองรับหลากหลายแพลตฟอร์ม
- มี DHCP Server ในตัว
- รองรับคุณสมบัติอย่าง Split Horizon, Geolocation routing, Dynamic DNS, และ Zone transfers
- มี Web UI ที่เข้าใจง่าย พร้อมการตั้งค่าเริ่มต้นที่สามารถใช้งานได้ทันที


## ขั้นตอนการติดตั้ง Technitium DNS บน Linux Ubuntu 22.04 LTS

ข้อกำหนดขั้นต่ำสำหรับสภาพแวดล้อมที่คุณจะติดตั้ง Technitium:

- CPU: 2 Core
- Memory: 2 GB
- Storage: 20 GB
- OS: Linux Ubuntu 22.04 LTS

<Steps>
### อัปเดต Repository

```bash copy
sudo apt update && sudo apt upgrade -y
```

### ติดตั้ง Technitium DNS

ใช้คำสั่ง `curl` เพื่อดาวน์โหลดและรันสคริปต์การติดตั้งอย่างเป็นทางการจาก Technitium

```bash copy
curl -sSL https://download.technitium.com/dns/install.sh | sudo bash
```

### การตั้งค่า Technitium DNS ครั้งแรก

- **เข้าสู่หน้าเว็บจัดการ Technitium DNS Server**

```
http://<ip_address>:5380
```

> อย่าลืมเปลี่ยน `<ip_address>` เป็น IP ของเครื่องเซิร์ฟเวอร์คุณ

- **ตั้งค่ารหัสผ่าน Administrator**<br/>
  เมื่อล็อกอินเข้าสู่ระบบครั้งแรก Technitium DNS จะบังคับให้คุณ **ตั้งค่ารหัสผ่านสำหรับบัญชีผู้ดูแลระบบ (admin)**

![technitium-install-1](/server/technitium/technitium-install-1.png)

- **ตั้งค่าหน้าเว็บเพื่อความปลอดภัย (Web Service Settings)**<br/>
  การตั้งค่าในส่วนนี้จะช่วยเพิ่มความปลอดภัยให้กับการเข้าถึงหน้าเว็บจัดการผ่าน HTTPS เพื่อให้การสื่อสารระหว่างเบราว์เซอร์ของคุณกับเซิร์ฟเวอร์ถูกเข้ารหัส

  - ไปที่เมนู `Settings`

  - เลือกเมนูย่อย `Web Service`

  - ในส่วน `HTTPS Options` ให้คุณติ๊กถูกที่ช่องเหล่านี้
    - ☑️ `Enable HTTPS`: เพื่อเปิดใช้งานการเชื่อมต่อที่เข้ารหัสแบบ HTTPS
    - ☑️ `Enable HTTP to HTTPS Redirection`: เพื่อให้การเข้าถึงผ่าน HTTP ถูกเปลี่ยนเส้นทางไปยัง HTTPS โดยอัตโนมัติ
    - ☑️ `Use A Self Signed TLS Certificate When TLS Certificate File Path is Unspecified`: เพื่อให้ Technitium สร้าง Self-Signed Certificate ให้เองอัตโนมัติ ในกรณีที่คุณยังไม่มีไฟล์ Certificate ของตัวเอง

    ![tehcnitium-install-2](/server/technitium/tehcnitium-install-2.png)

- เมื่อตั้งค่าทั้งหมดเสร็จเรียบร้อยแล้ว ให้เลื่อนหน้าจอลงมาที่ด้านล่างสุด แล้วคลิกปุ่ม `Save Settings`

</Steps>
