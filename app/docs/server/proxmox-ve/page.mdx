# Proxmox VE

Proxmox VE (Proxmox Virtual Environment) คือแพลตฟอร์มการจัดการเซิร์ฟเวอร์แบบเสมือนจริง (Server Virtualization Management Solution) แบบโอเพนซอร์ส (Open Source) ที่ทรงพลังและใช้งานง่าย

อธิบายง่ายๆ คือ Proxmox VE เปลี่ยนเครื่องคอมพิวเตอร์เซิร์ฟเวอร์จริง ๆ ของคุณให้กลายเป็น "โฮสต์" ที่สามารถรันระบบปฏิบัติการและแอปพลิเคชันเสมือนจริงได้หลายตัวพร้อมกัน หรือที่เรียกว่า Virtual Machine (VM) และ Container (LXC) โดยใช้ทรัพยากรของเซิร์ฟเวอร์จริงร่วมกันอย่างมีประสิทธิภาพ

## ขั้นตอนการติดตั้ง Proxmox VE 7.4

<Steps>

### เริ่มต้นการติดตั้ง Proxmox VE

หลังจากสร้าง Bootable USB ของ Proxmox VE และบูตเครื่องแล้วจะพบหน้าจอเริ่มต้นการติดตั้ง

- เลือก `Install Proxmox VE`
- กดปุ่ม `Enter` เพื่อดำเนินการต่อ

![proxmox-setup-1](/server/proxmox/proxmox-setup-1.png)

### ยอมรับข้อตกลงการใช้งาน

อ่านข้อตกลงการใช้งาน (End-User License Agreement) เพื่อทำความเข้าใจเงื่อนไขการใช้งาน

- คลิกปุ่ม `I agree` เพื่อยอมรับข้อตกลงและไปยังขั้นตอนถัดไป

![proxmox-setup-2](/server/proxmox/proxmox-setup-2.png)

### เลือก Storage สำหรับการติดตั้ง

- เลือกดิสก์ที่ต้องการ

- คลิกปุ่ม `Next` เพื่อดำเนินการต่อ

![proxmox-setup-3](/server/proxmox/proxmox-setup-3.png)

### กำหนด Time Zone

- Country: `Thailand`

- Time Zone: `ระบบจะเลือก Time zone จากช่อง Country โดยอัตโนมัติ`

- Keyboard Layout: `U.S. English`

- คลิกปุ่ม `Next`

![proxmox-setup-4](/server/proxmox/proxmox-setup-4.png)

### กำหนดรหัสผ่านและอีเมลผู้ดูแลระบบ

- Password: `กรอกรหัสผ่านที่คุณต้องการใช้สำหรับบัญชีผู้ดูแลระบบ`
- Confirm Password: `กรอกรหัสผ่านซ้ำอีกครั้งเพื่อยืนยัน`
- Email: `กรอกอีเมลสำหรับแจ้งเตือนหรือการกู้คืน`

<Callout>หากไม่ต้องการใส่อีเมลส่วนตัว ให้ระบุเป็น `mail@example.local`</Callout>

![proxmox-setup-5](/server/proxmox/proxmox-setup-5.png)

### กำหนดค่าเครือข่าย

- Hostname: `ชื่อเครื่อง`
- IP Address: `ระบุ IP Address`
- Netmask: `ระบุ Subnet Mask ของเครือข่าย`
- Gateway: `ระบุ IP Address ของ Gateway`
- DNS Server: `ระบุ IP Address ของ DNS Server`

![proxmox-setup-6](/server/proxmox/proxmox-setup-6.png)

### ตรวจสอบและเริ่มการติดตั้ง

- คลิกปุ่ม `Install` เพื่อเริ่มกระบวนการติดตั้ง Proxmox VE

- รอจนกว่ากระบวนการติดตั้งจะเสร็จสมบูรณ์

![proxmox-setup-7](/server/proxmox/proxmox-setup-7.png)

### เข้าสู่ระบบครั้งแรก

- เปิดเว็บบราวเซอร์ของคุณ แล้วพิมพ์ `IP Address` ที่แสดงบนหน้าคอนโซล ตามด้วยพอร์ต `8006`

```url copy
https://<ip_address>:8006
```

- เข้าสู่ระบบด้วยบัญชีผู้ใช้ `root` และรหัสผ่านที่คุณตั้งไว้

</Steps>
