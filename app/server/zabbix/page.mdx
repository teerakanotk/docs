# Zabbix

## เตรียมความพร้อมก่อนการติดตั้ง

- **Processor:** 1 Core
- **Memory:** 1 GB
- **Storage:** 20 GB
- **Operating system:** Linux Ubuntu 22.04 (Jammy) LTS
- **IP Address:** Static

## ขั้นตอนการติดตั้ง Zabbix Server

<Steps>

### ติดตั้ง Zabbix

```bash
bash -c "$(curl -fsSL https://raw.githubusercontent.com/teerakanotk/ctsurin-docs/main/script/zabbix/install.sh)"
```

```bash

  /tmp/zabbix_install_<date_time>.log

========================================================

Executing...
```
