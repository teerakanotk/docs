# ประเภทของ DNS Record

## A (IPv4 Address Record)

- หน้าที่: ชี้ชื่อโดเมนไปยัง IPv4 Address

![technitium-record-1](/assets/technitium/technitium-record-1.png)

## AAAA (IPv6 Address Record)

- หน้าที่: ชี้ชื่อโดเมนไปยัง IPv6 Address

![technitium-record-2](/assets/technitium/technitium-record-2.png)

## CNAME (Canonical Name Record)

- หน้าที่: ชี้ชื่อโดเมนหนึ่งไปยังอีกชื่อโดเมนหนึ่ง (alias)

![technitium-record-3](/assets/technitium/technitium-record-3.png)

> อธิบาย:
>
> - `www.homelab.local` ใช้ DNS เดียวกับ `homelab.local`

## MX (Mail Exchange Record)

- หน้าที่: ระบุ Mail Server สำหรับรับอีเมลของโดเมน

![technitium-record-4](/assets/technitium/technitium-record-4.png)

> อธิบาย:
>
> - ค่า `reference` ยิ่งต่ำ ยิ่งมีความสำคัญสูงกว่า และใช้ในการจัดลำดับความสำคัญของเมลเซิร์ฟเวอร์
> - หากเซิร์ฟเวอร์หลักล่ม, ระบบจะยังสามารถส่งเมลผ่านเซิร์ฟเวอร์รองได้

## NS (Name Server Record)

- หน้าที่: ระบุว่าใครเป็น DNS Server ที่ดูแลโดเมนนั้น

![technitium-record-5](/assets/technitium/technitium-record-5.png)

## SOA (Start of Authority Record)

- หน้าที่: ระบุข้อมูลต้นทางของ DNS Zone เช่น admin, serial number, refresh time
- มีเพียง 1 รายการ ต่อ zone

![technitium-record-6](/assets/technitium/technitium-record-6.png)

## PTR (Pointer Record)

- หน้าที่: ใช้สำหรับ Reverse DNS lookup (IP -> ชื่อโดเมน)

![technitium-record-7](/assets/technitium/technitium-record-7.png)
